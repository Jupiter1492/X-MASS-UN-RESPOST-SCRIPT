# X-MASS-UN-RESPOST-SCRIPT
X (Twitter) Repost deleter. It is a short Java Script that deletes your reposts. It is slow, but gets the Job done.
How to use it:
Go to your Profile page on X.
Make sure you are on the Posts or Replies tab (wherever your reposts are visible).
Open the Developer Tools (F12 or Right Click -> Inspect).
Go to the Console tab, paste the code, and hit Enter.
Keep the tab open and active on your screen.
